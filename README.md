# Город.ТГ - Mini App Игра

Полноценная видеоигра на основе логики Telegram бота.

## Структура проекта

```
mini-app/
├── index.html          # Главная страница
├── game.js            # Основная игровая логика и Phaser конфигурация
├── styles.css         # Стили для UI
└── README.md          # Этот файл
```

## Технологии

- **Phaser 3** - игровой движок для создания 2D игры
- **Telegram WebApp API** - интеграция с Telegram
- **HTML5/CSS3/JavaScript** - базовая веб-технология

## Развертывание

### Вариант 1: GitHub Pages

1. Создай новый репозиторий на GitHub (или используй существующий)
2. Загрузи файлы из папки `mini-app/` в корень репозитория
3. В настройках репозитория: Settings → Pages
4. Выбери source branch (обычно `main`)
5. Сохрани
6. GitHub предоставит URL вида: `https://твое-имя.github.io/название-репо/`

### Вариант 2: Vercel (Рекомендуется)

1. Зарегистрируйся на https://vercel.com
2. Подключи GitHub репозиторий
3. Vercel автоматически определит настройки
4. Deploy - получишь URL вида: `https://твой-проект.vercel.app`

### Вариант 3: Netlify

1. Зарегистрируйся на https://www.netlify.com
2. Перетащи папку `mini-app/` в Netlify Drop
3. Получишь URL сразу

## Настройка в Telegram Bot

1. Открой @BotFather в Telegram
2. Выбери своего бота
3. `/mybots` → выбери бота → `Bot Settings` → `Menu Button`
4. Укажи URL твоего Mini App
5. Или используй команду:
   ```
   /newapp - создай новое приложение
   ```
